OFELIA Control Framework

OFELIA Control Framework (OCF) is a set of software tools for testbed management. It controls experimentation life-cycle; reservation, instantiation, configuration, monitoring and uninistantiation.

It currently supports the following:

  • Full software stack: frontend, clearinghouse and resources managers (AMs)
  • Support for Openflow resources, Virtual Machines (XEN) reservation and bootstraping and Emulab resources.
