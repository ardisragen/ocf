Help

OFELIA provides two sources of help:
  • OFELIA User Documentation: This wiki gathers all the required information about the facility and how to work with the OFELIA Control Framework from an experimenter point of view.
  • OFELIA HelpDesk: meant to provide experimenters a way to report and open trouble tickets if they found themselves in a situation where the OFELIA User Documentation is not enought.
