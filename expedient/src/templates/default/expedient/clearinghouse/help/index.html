Help

OFELIA provides diverse sources of help:

  • OFELIA user documentation

    Information about the facility and how to experiment with the OFELIA Control Framework.

  • OFELIA helpdesk

    Meant to provide experimenters a way to report and open trouble tickets when OFELIA User Documentation is not enough.

  • Users discuss mailing list

    Its goal is to trigger discussions between the users and the Island Managers regarding any conflict, probem or doubt that could not be solved with the Documentation. OFELIA users are automatically registered to this list.
